Adonis —— MariaDB 数据库基本操作

  2020-04-28   Node.js   adonis  评论

MariaDB

安装

sudo yum install MariaDB-server MariaDB-client

启动

service mysql start

设置root用户密码

mysqladmin -u root password "mypassword"

登录

mysql -uroot -p

Adonis

adonis支持 PostgreSQL，SQLite，MySQL，MariaDB，Oracle，MSSQL 数据库，默认为SQLite。在 config/database 中修改配置。

配置数据库

connection: Env.get('DB_CONNECTION', 'mysql'),

创建数据库模型（Model）

adonis make:model posts

创建迁移（migration）

数据库迁移是一种从代码创建，更改和删除数据库表而不是编写SQL查询的过程。

adonis make:migration posts

查看状态

adonis migration:status

执行迁移，创建表格

adonis migration:run

PM2

pm2 start server.js
